# ONDSA: a testing framework based on Gaussian graphical model for differential and similarity analysis of multiple omics networks

Omics Networks Differential and Similarity Analysis (ONDSA) framework tests for structural differences and similarities across multiple omics networks, specifically designed for continuous omics data. This framework leverages the Gaussian Graphical Model (GGM) to establish omics networks. This approach focuses on entry-wise comparisons of precision matrices across multiple nonoverlapping groups, introducing two asymptotically
independent test statistics to sequentially estimate differential and similar structures while adjusting for correlated effects in the false discovery rate (FDR) control procedures.
